### 完整描述

```markdown
# Porthole - 看见生活的窗口

我们很容易在脆弱的时候被屏幕吸引，但真正重要的是屏幕外的生活。

Porthole 通过画中画技术，在你的屏幕上开启一扇小窗，让生活的光透进来，温柔地提醒你：此刻，屏幕之外还有更重要的事。

## 核心功能

**时间感知**
- 实时时钟显示，让你意识到时间的流逝
- 番茄钟计时器，帮你专注当下任务

**专注房间**
- 创建或加入局域网内的专注房间
- 与朋友、同事一起专注，互相陪伴监督
- 语音通信功能，随时交流进度或休息闲聊
- 看到彼此的专注状态，形成正向压力

**温暖陪伴**
- 可爱的猫咪动画，在屏幕角落静静陪伴
- 提醒你：生活中还有需要你关注的小生命

**真实连接**
- 相机实时画面，看看周围的真实世界
- 视频循环播放，记录生活中的美好瞬间

**使用洞察**
- 追踪你回归生活的时间
- 用 iOS Screen Time 风格的统计，庆祝你回归现实的每一刻
- 正向反馈机制：回归生活越久，获得的鼓励越多

## 设计理念

Porthole 这个名字来自船舱的圆形窗户。当你在数字世界的海洋中航行时，这扇小窗让你看见外面的天空和海浪，提醒你：

- 时间在流逝
- 生活在继续
- 真实的世界就在屏幕之外

## 适合谁使用

- 想要减少屏幕时间的你
- 需要专注工作/学习的你
- 希望与朋友一起互相监督专注的你
- 追求数字健康生活方式的你

## 特色亮点

- 画中画技术：小窗悬浮在所有应用之上，温柔而不打扰
- 局域网专注房间：与身边的人一起专注，无需互联网
- 语音通信：专注时也能保持轻量连接
- 后台运行：即使锁屏也能追踪你的"回归生活"时间
- 隐私优先：所有数据仅存储在你的设备上，语音仅在局域网内传输
- 精美设计：遵循 iOS 设计规范，简洁优雅

## 使用场景

- 刷社交媒体时，时钟提醒你已经过了多久
- 工作学习时，番茄钟帮你保持专注
- 和朋友在同一空间工作时，创建专注房间一起专注
- 休息时，猫咪陪伴让屏幕时间更温暖
- 睡前，统计数据鼓励你早点放下手机

---

Porthole 不是要让你完全远离屏幕，而是帮你更有意识地使用它。

在屏幕上开一扇窗，让生活的光透进来。

现在就开始你的数字健康之旅吧。
```

## 审核说明（给苹果审核团队）

```
Porthole 使用画中画（Picture-in-Picture）技术，通过将 UIView 转换为视频流实现内容悬浮显示。

主要用途：
1. 帮助用户意识到屏幕使用时间
2. 提供专注工具（番茄钟）
3. 追踪用户回归生活的时间，鼓励数字健康
4. 局域网专注房间：用户可以与同一局域网内的朋友创建/加入专注房间，共同专注

VoIP 功能说明：
- 应用使用 VoIP 后台模式支持局域网内的专注房间语音通信功能
- 用户可以在专注房间内与其他成员进行实时语音交流
- 语音数据仅在局域网内点对点传输，不经过任何服务器
- 此功能帮助用户与朋友/同事保持轻量连接，互相监督专注状态

相机权限说明：
- 仅用于"相机实时画面"功能
- 用户可选择是否使用此功能
- 不会上传或存储任何相机数据

麦克风权限说明：
- 仅用于专注房间内的语音通信功能
- 用户可随时开启/关闭麦克风
- 语音数据仅在局域网内传输，不会上传至任何服务器

所有数据仅存储在用户设备本地，不涉及互联网服务器传输。
```
