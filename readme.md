

## Idea

我想实现一个用以解决沉迷手机问题的 ios app 。我的思路是，用户沉迷手机是因为app的界面让他们忽略了环境或者时间信息，就像赌场封闭了所有窗户那样，所以要解决沉迷，就是要提供一个窗户，或者提供一个在梦境里不和谐的内容。
在这个窗户里，可以展示时间、番茄钟、实景天气、机会成本等内容，主旨就是给他们提供一些额外的环境信息，帮助他们保持觉知。
在ios里一般我们无法实现这样的窗户，但是我想通过视频播放的悬浮窗口来实现。这样app内部实时生成用户选择的展示内容，然后用户可以在悬浮视频窗口里调整大小，也可以通过暂停等按钮发送控制事件。
由于目标是减少用户对于手机的使用，所以我也想统计各种内容作为悬浮窗口计算的时长，并提供一个统计视图。
